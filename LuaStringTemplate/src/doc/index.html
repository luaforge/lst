Introduction

LuaStringTemplate is a Lua port of the StringTemplate template library. It supports a subset of the StringTemplate 3.1 feature set, primarily those that make code generation easier. See Supported StringTemplate Features for a detailed list of what works and what doesn't.

Supported StringTemplate Features

The following StringTemplate features are supported:

  • Basic attribute.property reference
  • Angle brackets and dollar sign expression delimiters are supported in the basic template
  • Expression options
    • separator is supported,
    • null, wrap, anchor, and format are parsed, but not honoured,
  • Group files
    • implements and the inherits syntax is parsed but not honoured,
    • Basic maps and template definitions are supported
    • Angle brackets are currently the only supported separator
    • Assumes all group files end in .stg

This is obviously an incomplete list, and features will be added gradually over time.
